Взаимодействие: alert, prompt, confirm

Рассмотрим 3 функции браузера для взаимодействия с пользователем:

alert
- показывает сообщение.

alert("Hello");

prompt
- показывает сообщение и запрашивает ввод текста от пользователя. Возвращает напечатанный в поле ввода текст или null, если была нажата кнопка «Отмена» или Esc с клавиатуры.

let age = prompt('Сколько тебе лет?', 100);
alert(`Тебе ${age} лет!`); // Тебе 100 лет!
confirm
- показывает сообщение и ждёт, пока пользователь нажмёт OK или Отмена. Возвращает true, если нажата OK, и false, если нажата кнопка «Отмена» или Esc с клавиатуры. let isBoss = confirm("Ты здесь главный?");
alert( isBoss ); // true, если нажата OK

Все эти методы являются модальными: останавливают выполнение скриптов и не позволяют пользователю взаимодействовать с остальной частью страницы до тех пор, пока окно не будет закрыто.

На все указанные методы распространяются два ограничения:

Расположение окон определяется браузером. Обычно окна находятся в центре. Визуальное отображение окон зависит от браузера, и мы не можем изменить их вид. Такова цена простоты. Есть другие способы показать более приятные глазу окна с богатой функциональностью для взаимодействия с пользователем, но если «навороты» не имеют значения, то данные методы работают отлично.

Преобразование типов

Строковое преобразование

let value = true;
alert(typeof value); // boolean
value = String(value); // теперь value это строка "true"
alert(typeof value); // string

Численное преобразование

alert( "6" / "2" ); // 3, строки преобразуются в числа
let str = "123";
alert(typeof str); // string
let num = Number(str); // становится числом 123
alert(typeof num); // number

Консоль

В вашем браузере есть инструменты разработчика, среди которых — консоль, которая позволяет набирать JavaScript прямо на этой веб-странице.

В Mozilla Firefox, например, откройте инструменты разработчика с помощью Ctrl+Shift+K или F12

Найдите в инструментах вкладку Console и посмотрим, сработает ли это!

В консоли введите или вставьте нижеследующий код и нажмите Enter

alert('Привет мир!')

Числа

Как и любой другой язык программирования, JavaScript может обрабатывать цифры.

Эти числа могут быть большими или малыми, с десятичными знаками или без них, объединенными посредством числовых операций...

alert(9+5*3)

Размеры браузера

Ширину браузера можно получить с помощью JavaScript:

alert(window.innerWidth)

Cуществует и window.innerHeight, и window.scrollY (даёт текущую позицию прокрутки).

Объекты

window
— это JavaScript объект (ещё один тип данных).

innerWidth
— это свойство объекта window.

Мы использовали точку ., чтобы указать, что нам нужен innerWidth, который существует внутри объекта window

Объект JavaScript представляет собой тип данных, который содержит другие типы данных.

Например, window также содержит:

  • window.origin, который является строкой
  • window.scrollY, который является числом
  • window.location, который является объектом

Если window — это объект, содержащий location, который является другим объектом, это означает, что JavaScript поддерживает dложенность.

Свойством объекта может быть другой объект.

Так как window.location тоже объект, он имеет свои собственные свойства. Чтобы получить доступ к этим свойствам, просто добавьте еще одну точку . и имя свойства.

Например, свойство href существует:

alert(window.location.href)

Методы

Если свойство объекта является функцией, то оно называется методом.

На самом деле, функция alert(), которую мы использовали до сих пор, — это метод объекта window:

window.alert('Слава Богу!')

Поскольку window является объектом верхнего уровня в браузере, вы можете напрямую обращаться ко всем его свойствам и методам.

Вот почему location.href совпадает с window.location.href.

Или почему alert() эквивалентен window.alert().

Объекты полезны для группировки нескольких свойств под одним именем (и уровнем иерархии). Это похоже на дерево, где каждая ветка может иметь другие более мелкие ветви.

Массивы

Массив JavaScript — это тип данных, который может содержать несколько значений, как если бы они были в списке.

Давайте передадим массив 3 строки в функцию alert():

alert(['Как', 'ваши', 'дела'])

Массив может содержать любые значения: строки, числа, объекты, другие массивы:

alert([2 + 5, 'самурай', true])

Логический тип (boolean)

В то время как строки и числа имеют бесконечное количество возможных значений, логический тип может быть либо true, либо false.

Объединение функции alert() и 3-го элемента массива в одной строке делает наш код менее читаемым.

Что, если бы мы могли разбить их, переместив массив на отдельную строку?

Переменные

Мы можем записать массив в переменную.

Переменная — это контейнер, который хранит определенное значение. Она имеет имя (поэтому вы можете идентифицировать и повторно использовать её), и она имеет значение (так что вы можете изменить её позже). var my_things = [2+5, 'самурай', true];

my_things — это имя переменной

[2+5, 'самурай', true] — это значение переменной

Это означает, что my_things равен [2+5, 'самурай', true].

Теперь вернёмся к функции alert.

Поскольку теперь мы имеем два действия, нам нужно разделить их точкой с запятой ;, чтобы знать, где заканчивается одно и начинается другое.

var my_things = [2+5, 'самурай', true];
alert(my_things);

Сохранив массив в переменной, нам удалось сделать код более читабельным.

Хотя эта переменная содержит массив из нескольких элементов, мы можем захотеть иметь дело только с одним элементом массива.

Чтобы получить доступ к конкретному элементу массива, вам нужно знать его индекс (позицию) в массиве. Затем нужно обернуть этот индекс в квадратные скобки.

Какое значение будет отображаться?

var my_things = [2 + 5, 'самурай', true];
alert(my_things[1]);

Появился второй элемент. В программировании индексы начинаются с нуля 0.

Например, my_things имеет свойство length:

Но оказывается, что массивы — тоже объекты! Это означает, что массивы также имеют свойства и методы.

Например, my_things имеет свойство length:

var my_things = [2 + 5, 'самурай', true];
alert(my_things.length);

Массив содержит 3 элемента. Если вы добавите или удалите элементы в my_things, то значение length изменится соответствующим образом.

Хотя читабельность и свойства полезны, основной концепцией массива является то, что его можно редактировать.

var my_things = [2 + 5, 'самурай', true];
alert(my_things);
my_things = [2 + 5, 'самурай', true, 'ЛЮБОВЬ'];
alert(my_things);

Массив my_things теперь содержит 4 элемента.

Метод push() изменил массив, но другие методы просто возвращают значение.

var my_things = [2 + 5, 'самурай', true];
alert(my_things.includes('ниндзя'));

Метод includes() проверяет есть ли в массиве строка 'ниндзя'. Такой строки нет, поэтому метод возвратил false (тип данных boolean).

Обычно, логические значения (true или false) встречаются в результате вызова функции (например, includes()) или как результат сравнения.

Вот сравнение «больше»: alert(window.innerWidth > 400);

Если было показано true, значит окно вашего браузера сейчас шире 400 пикселей.

Так же, как + и *, символ > является оператором JavaScript.

Когда вы делаете такое сравнение с оператором «больше» >, вы получаете значение boolean.

Поскольку результат сравнения возвращает только 2 возможных результата (true или false), это полезно в тех случаях, код должен принять решение...

Условия

Условные выражения — одно из наиболее важных понятий в программировании. Они позволяют выполнять определенные команды только если соблюдены определенные условия. Эти условия могут, к примеру, основываться на:

  • авторизации пользователя (был ли пароль введен правильно?)
  • текущем состоянии (день или ночь?)
  • каком-либо значении (этот человек старше 18 лет?)

Давайте покажем окно предупреждения только если вы оказались на домене редактора этого документа.

if (window.location.hostname == 'seniorua.github.io') {
alert('Вы на домене Андрея. Добро пожаловать!');
} else alert('Вы на другом домене'); Здесь мы делаем сравнение. В Javascript для этого используется оператор == (два знака «равно»). Мы обработали случай, когда сравнение возвращает true. Для обработки противоположного случая, когда имя хоста не 'nicothin.pro', мы можем использовать оператор «не равно» !=: if (window.location.hostname != 'nicothin.pro') { alert('Пожалуйста возвращайся скорее!'); } Этот фрагмент кода ничего не запускает. Но что если мы хотели обрабатывать оба случая одновременно? Можно написать два оператора if, однако одно утверждение является полной противоположностью другого и мы можем объединить их с помощью else: if (window.location.hostname == 'nicothin.pro') {
alert('Добро пожаловать на мой домен! :)');
} else {
alert('Пожалуйста возвращайся скорее! ;)');
}
